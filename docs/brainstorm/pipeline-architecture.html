Implementation Roadmap

Phase 1
Week 1-2

MVP: Manual CLI + Discord Webhook

Get Gemini API key, create Discord webhook, build TypeScript scripts (generate.ts, discord.ts, cli.ts, templates.ts). Test with 10 topics. Refine prompts until quality matches the bamboo infographic standard. ~200 lines of code total.

Phase 2
Week 3-4

Automated + Content Calendar

Supabase content_calendar table. Scheduled task (wildarc-content-creator, 12:30 PM daily). Google Drive upload integration. Watchdog health checks for pipeline. Bulk-seed 30 days of topics.

Phase 3
Month 2

Instagram + Feedback Loop

Instagram Business Account + Meta API. Supabase Storage for public CDN URLs. Discord bot (upgrade from webhook) for reaction tracking. content_performance table for analytics.

Phase 4
Month 3+

Self-Improving Content Engine

Steward auto-feeds topics. Performance-based weighting. A/B testing (2 versions → Discord → track winner). Seasonal awareness. Community request queue. Website gallery.
